# SAARTHI AI PROJECT

*From Confusion to Clarity. From Goals to Results.*

SAARTHI AI is a Personal Growth Operating System. It acts as an accountability partner, goal tracker, decision coach, and career guide. 

## Features
- **Onboarding**: Captures your goals and creates roadmaps.
- **Goal Tracker**: Manages and tracks progress.
- **Daily Briefing**: Highlights your top 3 priorities.
- **Accountability**: Follows up on commitments.
- **Persistent Memory**: Uses local JSON file storage to remember your profile.

## How to run
1. Install dependencies: `pip install -r requirements.txt`
2. Start the app: `streamlit run app.py`
