# AI Assistant Bootcamp Projects

Two AI assistant projects built at the VEDAM AI Builders Bootcamp (June 14-15, 2026).

## Projects

### 1. Gym Trainer AI (Live Bootcamp Build)
Folder: AI_ASSISTANT_LIVE_BOOTCAMP_PROJECT/
A personal fitness AI assistant built live during the bootcamp session using Python, Ollama, Mistral, and Streamlit.

### 2. SAARTHI AI (Original Competition Project)
Folder: SAARTHI AI PROJECT/
A Personal Growth Operating System that acts as your mentor, strategist, and accountability partner. Named after Lord Krishna who served as Arjun's Saarthi (charioteer) in the Mahabharata — guiding through confusion to clarity.

## Tech Stack
- Python 3.x
- Streamlit (Web UI)
- Ollama (Local AI Model Runner)
- Models: Mistral, Qwen2.5, TinyLlama

## How to Run
1. Install: pip install ollama streamlit
2. Pull model: ollama pull qwen2.5:3b
3. cd into any project folder
4. Run: streamlit run app.py

## Current Status
MVP stage — core chat functionality working with local AI models. Future versions will include persistent memory, goal tracking dashboard, Groq API integration for better responses, and deployment as a public web app.
